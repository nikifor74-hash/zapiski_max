# MAX ToDo Bot

Простой бот для ведения списка задач (ToDo) в мессенджере MAX.

## Возможности

- Добавление задачи: `/add Текст задачи`
- Просмотр списка: `/list` (с inline-кнопками для отметки выполнения)
- Отметка о выполнении: `/done номер` или нажатие кнопки
